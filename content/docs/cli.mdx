# CLI

The Lunar Kit CLI helps you add and manage components in your React Native project.

## Installation

```bash
# Global installation (recommended)
pnpm add -g @lunar-kit/cli

# Or use npx without installing
npx @lunar-kit/cli [command]
```

## init

Initialize Lunar Kit in your existing React Native project.

```bash
lunar-kit init
```

### What it does

1. Creates `src/components/ui` directory
2. Sets up `lib/utils.ts` with helper functions
3. Installs required dependencies (clsx, tailwind-merge)
4. Creates `kit.config.json` configuration file

## add

Add components from the Lunar Kit registry to your project.

```bash
lunar-kit add <component-name...>
```

### Examples

```bash
# Add a single component
lunar-kit add button

# Add multiple components
lunar-kit add button card dialog input

# Add all available components
lunar-kit add --all
```

### Options

| Option | Description |
|--------|-------------|
| `--all` | Add all available components |
| `--path` | Custom path for components (default: `src/components/ui`) |

### What happens

1. Downloads component source files
2. Resolves and installs registry dependencies
3. Installs required npm packages
4. Updates barrel exports if present

## Configuration

Lunar Kit stores configuration in `kit.config.json`:

```json title="kit.config.json"
{
  "typescript": true,
  "componentsPath": "src/components",
  "packageManager": "pnpm"
}
```

### Options

| Option | Default | Description |
|--------|---------|-------------|
| `typescript` | `true` | Use TypeScript for components |
| `componentsPath` | `src/components` | Where to store components |
| `packageManager` | `pnpm` | Package manager to use |

## Workflow Examples

### Starting a New Feature

```bash
# 1. Create a module for the feature
lunar-kit module checkout

# 2. Add required UI components
lunar-kit add button card input form

# 3. Create custom components
lunar-kit gen scoped checkout CheckoutForm
lunar-kit gen scoped checkout OrderSummary
lunar-kit gen scoped checkout PaymentMethod
```

### Adding Components to Existing Project

```bash
# 1. Initialize Lunar Kit
lunar-kit init

# 2. Add commonly used components
lunar-kit add button text card input

# 3. Add more as needed
lunar-kit add dialog bottom-sheet tabs
```

### Full Component Setup

```bash
# Add all components at once
lunar-kit add --all
```

## Troubleshooting

### "Command not found"

```bash
# Ensure global installation
pnpm add -g @lunar-kit/cli

# Or use npx
npx @lunar-kit/cli add button
```

### "Registry not found"

The CLI uses `@lunar-kit/core` for the component registry. Make sure it's installed:

```bash
pnpm add @lunar-kit/core
```

### Component dependencies missing

Run the add command again, it will install missing dependencies:

```bash
lunar-kit add button
```

### Path issues

Check your `kit.config.json` has the correct paths:

```json
{
  "componentsPath": "src/components"
}
```

## Tips

1. **Use npx for one-time use** â€” No need to install globally if you only use it occasionally

4. **Check dependencies** â€” Some components require others (e.g., `date-picker` requires `calendar`)
